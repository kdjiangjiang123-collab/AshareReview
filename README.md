# 📈 A股复盘助手

交易者的 AI 复盘伙伴 — 盘中快速研判 + 盘后深度复盘 + 交易持仓管理。

## ✨ 功能

| 模式 | 说明 |
|------|------|
| 📡 **盘中研判** | 实时指数 + 大盘 AI 快扫 + 持仓深度分析 |
| 📝 **盘后复盘** | 市场全景数据 + AI 三层分析（宏观/微观/情景推演）|
| 💼 **交易持仓** | 交易录入 + 持仓管理 + 实时行情 + 个股 K 线图 |

## 🚀 快速开始

### 1. 安装 Python 3.10+

### 2. 下载本项目（Code → Download ZIP），解压

### 3. 双击 `启动复盘助手.bat`

### 4. 首次使用会弹出配置页

粘贴你的 DeepSeek API Key（[免费获取](https://platform.deepseek.com/api_keys)，一个月几块钱），点保存即可。

## 📊 数据源

全部免费，无需额外 API Key：

| 数据 | 来源 |
|------|------|
| 实时行情 | 腾讯接口 |
| K 线历史 | Ashare（新浪+腾讯双引擎）|
| 北向资金 / 板块 | akshare |
| 技术指标 | MyTT（MACD / KDJ / BOLL / RSI / 量比 / 均线）|

## 🧠 AI 引擎

[DeepSeek](https://deepseek.com) — chat 模型快速分析，reasoner 模型深度推理。

## 🔒 隐私

所有交易数据、持仓信息本地 SQLite 存储，不上传任何服务器。

## 📁 项目结构

```
├── app.py               # Streamlit 主入口

├── db/                  # 数据库操作
├── ui/                  # 前端页面
├── config/              # 配置
└── data/                # 本地数据库（不上传 GitHub）
```
